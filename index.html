Pneumonia Detection

A binary classification model detecting Pneumonia across 6,000+ chest X-rays. It fine-tunes a VGG16 architecture using image augmentation to handle class imbalance and overfitting, reaching a 93% Accuracy and 92% F1-score on test data.

  • Python
  • TensorFlow
  • Keras

Fraud Detection

A binary classification pipeline engineered to detect fraud from highly skewed data. Through strategic resampling and hyperparameter fine-tuning on over-sampled distributions, it countered severe class imbalance to secure a 99.81% Minority Recall and a 2.3x F1-score increase from 0.31 to 0.73.

  • Python
  • Scikit-learn
  • XGBoost
